# The English Path

Referencia de gramática inglesa en español, pensada para ser clara y directa. Una app web de una sola página, sin dependencias, que funciona en cualquier navegador moderno.

## Contenido

**30 temas** organizados en 4 secciones:

- **Tiempos verbales** — Present Simple, Continuous, Perfect, Past Simple, Past Perfect, Future Will / Going to, y más (11 temas)
- **Verbos modales** — Can/Could, May/Might, Must/Have to, Should, Modal Perfects, Used to/Would (6 temas)
- **Condicionales** — Zero, First, Second, Third, Mixed Conditionals, Wish/If only (6 temas)
- **Gramática esencial** — Passive Voice, Articles, Reported Speech, Relative Clauses, Gerunds & Infinitives, Comparatives, Quantifiers (7 temas)

Cada tema incluye:
- Explicación de cuándo se usa
- Estructura con fórmulas visuales
- Marcadores de tiempo y contexto
- Ejemplos con traducción al español
- Errores comunes a evitar
- Notas de registro (cuando aplica)

## Uso

Abre `index.html` directamente en el navegador. No requiere servidor ni instalación.

```
open index.html
```

## Tecnología

- HTML, CSS y JavaScript vanilla — sin frameworks ni dependencias
- Tipografías: [Fraunces](https://fonts.google.com/specimen/Fraunces) y [Lexend](https://fonts.google.com/specimen/Lexend) vía Google Fonts
- Diseño mobile-first, optimizado para pantallas pequeñas
